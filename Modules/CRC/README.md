#CRC-Check-Library

Features   
• Support CRC – 7  
• Support CRC – 8  
• Support CRC – 8  / ROHC  
• Support CRC – 16 / USB  
• Support CRC – 16 / MODBUS  
• Support CRC – 16 / XMODEM / ZMODEM / ACORN  
• Support CRC – 16 / CCITT-FALSE   
• Support CRC – 32  
• Support CRC – 32 / IEEE802-3(MPEG-2)  
• Support three calculated speed: fast, medium, slow
